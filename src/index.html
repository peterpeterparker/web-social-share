Web Social Share

Web Social Share is a Web Component to share urls and text on social platforms.

Currently supported:

Twitter, Facebook, Pinterest, Linkedin, Reddit, Email, Copy to clipboard, Hackernews, Whatsapp, Telegram, Openchat and DSCVR

Most modern browsers implement the Web Share API.
If not supported or not well suited, then this component can help.
